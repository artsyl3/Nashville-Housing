# 🏠 Nashville Housing Data Cleaning Project (SQL)

## 📌 Project Overview

This project focuses on cleaning and transforming the **Nashville Housing dataset** using Microsoft SQL Server (SSMS).

The goal of this project is to demonstrate practical SQL data cleaning skills that are commonly required in real-world data analyst roles.

The dataset contains property sale information such as:

* Sale Price
* Sale Date
* Property Address
* Owner Address
* Parcel ID
* Sold As Vacant status

---

## 🎯 Objectives

During this project, the following data cleaning tasks were performed:

* Imported raw Excel data into SQL Server
* Standardized date formats
* Populated missing property addresses using self-joins
* Split address columns into separate fields (Street, City, State)
* Converted Y/N values to Yes/No using CASE statements
* Identified and removed duplicate records using CTE and ROW_NUMBER()
* Dropped unused columns to prepare a cleaner dataset
